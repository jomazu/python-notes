# Terminal Commands
[HOME](../README.md)

## Macbook

### Python Interactive Shell  

- EXIT the python interactive shell  
```ctrl + z```  

### VS Code
- OPEN/CLOSE terminal  
```ctrl + backtick```  

- Install PYLINT for linting in Python  
```shift + cmd + p```  
Type LINT or PYTHON to make sure it is on and/or active  

- Show Linting issues  
```shift + cmd + m```
